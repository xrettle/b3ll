## üöÄ Getting Started

### Prerequisites

- Node.js 18+ 
- npm or bun

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/aesthetic-bell-timer.git
   cd aesthetic-bell-timer
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   bun install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   Navigate to [http://localhost:3000](http://localhost:3000)
